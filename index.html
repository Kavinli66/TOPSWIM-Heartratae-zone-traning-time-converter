主项提示

计算说明

本软件以 200 米与 400 米自由泳测试成绩为输入，估算 CSS 阈值配速，并以 CSS 为中心生成五大训练配速区间。
CSS = (400m 成绩 - 200m 成绩) / 2
配速区间基于 CSS 估算，并结合最大心率百分比进行训练强度提示。该结果适合作为训练参考，实际训练仍需结合心率、主观用力感、技术状态和恢复情况进行校准。

免责声明 / 使用范围

本工具用于游泳训练配速参考，不构成医学建议、诊断意见或正式运动处方。实际训练应结合教练指导、个人身体状态、心率监测、主观用力感、技术表现和恢复情况进行调整。
