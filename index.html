主项提示

计算说明

本软件要求输入 50 米、200 米与 400 米成绩。短距离主项采用 50/400 CSS，更强调通过 50 米极限冲刺剥离无氧速度贡献；长距离主项采用 200/400 CSS，更接近长距离巡航配速；中距离主项取两种 CSS 的平均。
50/400 CSS 配速 = (400m 成绩 - 50m 成绩) / 3.5
200/400 CSS 配速 = (400m 成绩 - 200m 成绩) / 2
配速区间按 CSS 速度百分比换算：一区约 65–75%，二区约 75–85%，三区约 85–92%，四区约 92–100%，五区约 100–108%。该结果适合作为训练参考，实际训练仍需结合心率、主观用力感、技术状态和恢复情况进行校准。

本工具用于游泳训练配速参考，不构成医学建议、诊断意见或正式运动处方。实际训练应结合教练指导、个人身体状态、心率监测、主观用力感、技术表现和恢复情况进行调整。
