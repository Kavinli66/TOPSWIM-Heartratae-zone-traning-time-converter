主项提示

计算说明

本软件以 50 米、200 米与 400 米自由泳测试成绩为输入，按主项距离选择 CSS 阈值配速估算方式，并以 CSS 为中心生成五大训练配速区间。
短距离（50/100 米）：CSS 配速 = (400m 成绩 - 50m 成绩) / 3.5
中距离（200/400 米）：CSS 配速 = 50/400 法与 200/400 法加权平均
长距离（800/1500 米）：CSS 配速 = (400m 成绩 - 200m 成绩) / 2
为避免短距离运动员的冲刺能力把有氧区间拉得过快，短距离模式下第一区至第三区不会快于 200/400 法生成的对应区间；第五区上区间也设置了速度保护，避免配速过快。该结果适合作为训练参考，实际训练仍需结合心率、主观用力感、技术状态和恢复情况进行校准。
